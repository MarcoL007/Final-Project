# Final-Project
Number Guessing game 

## Project Description
A single player, terminal-based game where the computer generates a random number and the player attempts to guess it based on only two feed backs, either "Too High" or "Too Low".

## Features
- Difficulty levels: Three different game modes Easy(1-10), Medium(1-50), and Hard(1-100).
- Attempt limit: Each level has a specific number of allowed guesses.
- Input Validation: The program ensures only numeric inputs are accepted to prevent crashes.
- Replay Mode: Players can restart the game immediatly after they finish their current game

  ## Instructions to Run
  1. Have pyhton installed
  2. Download the .py file
  3. Run the following command in your terminal:
     'python "Final Project(CMPSC132)-Guess The Number.py"'
